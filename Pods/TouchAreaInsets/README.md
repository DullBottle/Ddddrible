# TouchAreaInsets

Touch area insets for UIView.

## At a Glance

The code below expands button's touch area for 20px.

```swift
let button = UIButton()
button.touchAreaInsets = UIEdgesInsets(top: 20, left: 20, bottom: 20, right: 20)
```

## Installation

```ruby
pod 'TouchAreaInsets'
```
